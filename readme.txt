A professional-grade bridge between Gravity Forms and the Palm Beach County GIS (ArcGIS) Property Information Table.

== Description ==

This plugin allows South Florida HOAs and businesses to verify property ownership in real-time. By connecting directly to the Palm Beach County Property Appraiser's data, it cross-references form submissions against official county records.

= Key Features =
* **Real-time Verification**: Automatically queries the PBC GIS database upon form submission.
* **Fuzzy Name Matching**: Compares resident-provided names against official county records (Owner 1 and Owner 2).
* **Condo Mode**: Optimized for HOA/Condo environments with support for static base addresses and unit number mapping.
* **Legacy Migration Utility**: Built-in tool to split legacy "Full Address" strings into separate House Number and Street Name fields.
* **Admin Sidebar Integration**: View match status, official owner names, and PCN directly within the entry detail page.
* **Dynamic Merge Tags**: Use `{pbc_raw_data}` in notifications to send a branded, color-coded result box to staff.
* **Bulk Processing**: Re-verify entries at once from the Gravity Forms entry list.

== Installation ==

1. Upload the `gf-pbc-gis` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the specific Gravity Form you wish to use.
4. Go to **Settings > PBC Verification**.
5. Enable the service and map your form fields (House Number, Street Name, and Resident Name).

== Merge Tags ==

= {pbc_raw_data} =
This is the primary output tag for emails. It renders a clean HTML table with the Match Status, Official Owners, and a direct link to the Property Appraiser (PAPA) website. If a name mismatch occurs, it provides a "Search by Name" fallback button.

== Migration Utility ==

For legacy sites with a single address field:
1. Enable **Migration Mode** in the main PBC Settings tab.
2. Click the new **Migration Utility** tab that appears in the form settings menu.
3. Map your "Full Address" source to your new "Number" and "Street" targets.
4. Run the process. Standardized suffixes (Dr, St, Ln) will be applied automatically.
5. Turn off Migration Mode when finished.
